# Tæknileg skjöl

Þessi mappa safnar saman tæknilegum skjölum um vefinn. Markmiðið er að tæknilegt efni sé á einum stað, frekar en dreift um alla skjalamöppuna.

Hún er fyrst og fremst fyrir þá sem þurfa að skilja uppbyggingu, þróunarumhverfi, deploy eða AI-vinnulag í repo-inu.

## Í þessari möppu

- `vefur-uppbygging.md` útskýrir hvernig vefurinn er settur saman.
- `throunarumhverfi-fyrir-ai-vinnu.md` lýsir þróunar- og preview vinnulagi.
- `foundry-agent-instructions.md` geymir leiðbeiningar fyrir sérhæfðan agent.
- `vercel-deploy.md` útskýrir deploy og preview á Vercel.

## Mikilvæg afmörkun

Þótt tæknilegu skjölin séu nú saman á einum stað þarf sjálfur vefurinn áfram að halda Next.js uppbyggingu inni í `web/`, til dæmis `web/app/`, `web/public/`, `web/components/` og stillingaskrár eins og `web/package.json` og `web/next.config.mjs`.
