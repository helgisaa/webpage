# Heimildir, brand og bakgrunnur

Þessi mappa safnar saman öllu því sem er gagnlegt sem bakgrunnur, heimild eða reference, en þarf ekki að vera sýnilegt í rót repo-ins fyrir daglega efnis- eða vefvinnu.

## Í þessari möppu

- `brand-manual/` geymir brandreglur, litaviðmið, font og sjónrænan grunn.
- `saa-design-audit/` geymir audit, samanburð og upprunaleg assets sem notuð voru sem viðmið.
- `saa_greining_nyr_vefur.md` geymir upprunalega greiningu og bakgrunn fyrir verkefnið.

## Hvenær á að fara hingað?

Farðu hingað þegar þú þarft að skilja uppruna ákvarðana, bera saman ásýnd, skoða brandreglur eða finna eldri vinnugögn.

Ef þú ert að breyta vefnum sjálfum eða contenti fyrir notendur áttu líklega ekki að byrja hér.
